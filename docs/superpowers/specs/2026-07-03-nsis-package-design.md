# NSIS 安装包配置设计

## 概述

为 time-master（时间管理工具）配置 Windows NSIS 安装包，支持用户自定义安装目录，提供标准的安装体验。

## 需求

- 生成 Windows NSIS 安装包（.exe）
- 默认安装到 `E:\时间管理工具`，用户安装时可自由选择路径
- 安装包文件名：`时间管理工具_Setup.exe`（不带版本号）
- 安装流程包含：欢迎 → 选择目录 → 快捷方式选项 → 安装 → 完成
- 创建桌面快捷方式（可选）
- 创建开始菜单快捷方式
- 安装完成后可选"立即运行"
- 安装界面语言：简体中文

## 技术方案

### 方案选择：Tauri 内置 NSIS 支持 + 自定义模板

采用 Tauri v2 内置的 NSIS 打包机制，配合自定义 NSIS 模板实现安装路径、快捷方式等自定义配置。

### 改动范围

涉及两个文件：

- `src-tauri/tauri.conf.json` — 修改 `bundle` 节点配置
- `src-tauri/nsis/template.nsi` — 新增自定义 NSIS 安装模板

### 具体配置

#### tauri.conf.json

```json
{
  "bundle": {
    "active": true,
    "targets": ["nsis"],
    "icon": [
      "icons/32x32.png",
      "icons/128x128.png",
      "icons/128x128@2x.png",
      "icons/icon.icns",
      "icons/icon.ico"
    ],
    "windows": {
      "nsis": {
        "installMode": "currentUser",
        "languages": ["SimpChinese"],
        "installerIcon": "icons/icon.ico",
        "template": "./nsis/template.nsi"
      }
    }
  }
}
```

### 配置字段说明

| 字段 | 值 | 说明 |
|------|-----|------|
| `targets` | `["nsis"]` | 仅生成 NSIS 安装包，不生成 MSI |
| `installMode` | `"currentUser"` | 当前用户安装模式 |
| `languages` | `["SimpChinese"]` | 安装界面语言：简体中文 |
| `installerIcon` | `icons/icon.ico` | 安装包图标 |
| `template` | `./nsis/template.nsi` | 自定义 NSIS 模板路径 |

#### 自定义 NSIS 模板功能

| 功能 | 实现方式 |
|------|---------|
| 默认安装路径 | `E:\时间管理工具`（用户可在安装时修改）|
| 桌面快捷方式 | 安装时自动创建，卸载时自动删除 |
| 开始菜单快捷方式 | 安装时自动创建到 `开始菜单 → 时间管理工具` |
| 卸载入口 | 控制面板"程序和功能"可卸载 |
| 安装完成选项 | 可选"立即运行 时间管理工具" |
| 安装包输出文件 | Tauri 自动生成 `时间管理工具_0.1.0_x64-setup.exe` |

### 安装流程

| 步骤 | 页面 | 用户操作 |
|------|------|---------|
| 1 | 欢迎页 | 查看软件信息，点击"下一步" |
| 2 | 安装目录选择 | 默认 `E:\时间管理工具`，可浏览修改 |
| 3 | 安装进度 | 等待安装完成 |
| 4 | 完成 | 勾选"运行 时间管理工具"，点击"完成" |

### 安装后效果

- **快捷方式**：开始菜单 → 时间管理工具 → 时间管理工具；桌面 → 时间管理工具（快捷方式）
- **卸载**：控制面板 → 程序和功能 → 时间管理工具（右键卸载）
- **数据存储**：数据文件保存在安装目录（`E:\时间管理工具\`），E 盘非系统盘，读写不受限

## 不涉及

- 数字签名（当前不处理）
- MSI 安装包
- macOS/Linux 打包
- 自动更新
- 数据存储路径改造

## 验收标准

1. `npm run tauri build` 成功生成 `.exe` 安装包
2. 安装包文件包含产品名和版本号（如 `时间管理工具_0.1.0_x64-setup.exe`）
3. 运行安装包，默认安装路径为 `E:\时间管理工具`
4. 允许用户在安装时修改安装路径
5. 安装后桌面快捷方式和开始菜单快捷方式正确创建
6. 可从控制面板正常卸载
7. 安装界面为简体中文
